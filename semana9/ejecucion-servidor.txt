Al ejecutar el comando **node server.js**, el servidor inicia correctamente y queda esperando solicitudes de los usuarios. En la terminal aparece el mensaje **"Servidor ejecutándose en puerto 3000"**. Al visitar la ruta **/estado**, el navegador muestra una respuesta en formato JSON indicando que el servidor está funcionando y que el servicio corresponde a una API comunitaria.

{
"estado": "Servidor funcionando",
"servicio": "API comunitaria"
}
