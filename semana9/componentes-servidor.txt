Sistema operativo:
Es el software que administra el hardware y permite ejecutar aplicaciones. También gestiona los recursos del computador.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y la forma en que responde a las solicitudes de los usuarios.

Puerto:
Es el canal de comunicación que utiliza el servidor para recibir solicitudes. En esta actividad se usa el puerto 3000.

Logs:
Son registros que almacenan información sobre la actividad del servidor. Ayudan a detectar errores y supervisar el funcionamiento del sistema.
